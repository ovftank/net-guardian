## 📥 Hướng dẫn cài đặt

### 1. Clone repository

```bash
git clone https://github.com/ovftank/net-guardian.git
cd net-guardian
```

### 2. Cài đặt thư viện

```bash
pip install -r requirements.txt
```

### 3. Chạy ứng dụng

```bash
python main.py
```

## 🔨 Build ứng dụng

Để tạo file thực thi (.exe), chạy:

```bash
build.bat
```

> 💡 File .exe sẽ được tạo trong thư mục `dist`
